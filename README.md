In this work I use html/css, javascript, python, sqlalchemy. In python I used Flask framework. 
This website can: 
1)Do shodan test
2)Scan ports.
3)Analyze HTTP Headers.
4)Search CGI and JS scripts.
It also saves scan history.
Dark mode is also included.
